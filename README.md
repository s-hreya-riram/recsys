# Deep Learning for Recommender Systems

DSS5104 Assignment — Shreya Sriram (A0327236E)

Implements and compares four recommender system approaches across two datasets:

| Model | Feedback type |
|-------|--------------|
| Popularity baseline | Both |
| Matrix Factorization (SVD / ALS) | Explicit / Implicit |
| Neural Collaborative Filtering (NCF) | Both |
| Two-Tower (BPR loss) | Both |

Datasets: **MovieLens 100K** (explicit) and **Amazon Digital Music 2018 5-core** (implicit).

## Data

### MovieLens 100K
Download `ratings.csv` from [grouplens.org/datasets/movielens/latest](https://grouplens.org/datasets/movielens/latest/) and place at `data/raw/movielens/ratings.csv`.

### Amazon Digital Music 2018 5-core
Download the 2018 5-core Digital Music review file from [nijianmo.github.io/amazon](https://nijianmo.github.io/amazon/) (newline-delimited JSON) and place at `data/raw/amazonmusic/ratings.json`.

## Reproducing results

### 1. Preprocess

```bash
python src/data/preprocessing.py --dataset movielens
python src/data/preprocessing.py --dataset amazonmusic
```

Expected output:
```
# MovieLens
Train: 77,464 | Val: 9,696 | Test: 9,904
Cold-start users: 145

# Amazon Music
Train: 124,807 | Val: 20,812 | Test: 23,622
Cold-start users: 89
```

### 2. Run the full pipeline

```bash
python src/pipeline.py --dataset movielens --skip-preprocess
python src/pipeline.py --dataset amazonmusic --skip-preprocess
```

Results are saved to `results/{dataset}/metrics.csv`.

### 3. Hyperparameter tuning (optional — best configs already in pipeline.py)

```bash
python src/tune_mf.py --dataset movielens
python src/tune_mf.py --dataset amazonmusic
python src/tune_ncf.py --dataset movielens
python src/tune_ncf.py --dataset amazonmusic
python src/tune_two_tower.py --dataset movielens
python src/tune_two_tower.py --dataset amazonmusic
```

Tuning results are saved to `results/{dataset}/{model}_tuning.csv`.

### 4. Generate convergence plots

```bash
python src/plot_convergence.py
```

Figures saved to `results/figures/`.

## Best hyperparameters

| Model | Dataset | Best config | Val NDCG@10 |

| NCF | MovieLens | emb=32, layers=[128,64,32], lr=0.001, n_neg=8 | 0.0708 |
